What We Offer
Built on Real Data.
Powered by Science.
🛰️
NASA POWER API

Multi-year historical solar irradiance data from NASA's POWER dataset, averaged to give you reliable baseline metrics for any location on Earth.

🌦️
Live Weather Fusion

Real-time temperature and cloud cover from OpenWeather API, dynamically adjusting your solar efficiency calculation based on today's actual conditions.

📈
7-Day Forecast

Trend-based energy generation predictions for the next 7 days, using irradiance baselines with noise modelling to simulate realistic variability.

💡
Cost Savings Engine

Estimates monthly electricity savings in ₹ based on your generated kWh and current average Indian electricity rates. Plan your investment smarter.

🗺️
Interactive Map

Leaflet-powered interactive map interface. Click anywhere to place a pin and instantly receive full solar analytics for that precise coordinate.

⚡
Instant Analysis

Flask-powered REST API backend returns comprehensive solar metrics in seconds. From click to insight — no waiting, no complexity.

About the Project
Open Source.
Community Driven.

SolarIQ is an academic and open-source project built to democratize access to solar potential data across India. Built with Python (Flask), NASA POWER, and OpenWeatherMap APIs. Designed for researchers, engineers, and solar entrepreneurs who need accurate, location-specific solar intelligence.
